Overview
========

VERA is a tool designed to help researchers collect data from XR experiments. It is made up of both a web client and a Unity plugin, which work together to aid in collecting data from your experiments.

This documentation is designed to guide you through the process of installing and using VERA for your experiments, allowing you to make use of both the web client and Unity plugin. This process is split into various major sections, each outlined below.

`Installation <installation.html>`_:
  In order to use VERA, you will need to install Unity and import the VERA Unity package. 
  This section guides you through the installation process, assuming you are starting from ground zero.
  This includes the installation of Unity, the download and import of the VERA Unity plugin, and an overview of how to ensure necessary dependencies are installed.

`Setup <setup.html>`_:
  After installing VERA, you will need to set up your Unity project to work with VERA. 
  This section guides you through the process of logging into the VERA portal, authenticating your Unity project, and ensuring that your project is properly configured to communicate with the VERA portal.

`Demo <demo.html>`_:
  To test your setup, you can run a demo scene that logs data to the VERA portal. 
  This demo scene includes a variety of scripts that demonstrate how to log data using the VERA Logger, and how to record to your particular file types.
  This section guides you through opening the demo scene, running the demo to log data, viewing the results in the VERA portal, and understanding the  demo scripts to use their general concepts for your own experiments.

`Unity Plugin <unity_overview.html>`_:
  This section provides an overview of the VERA Unity plugin, including its features, capabilities, and how it integrates with the VERA portal.
  This section is proceeded by more detailed sections on individual components of the Unity plugin, including 
  `Logging Data <logging_data.html>`_, `Building for WebXR <webxr.html>`_, and the `VLAT Accessibility Tools <vlat.html>`.

`VERA Locomotion Accessibility Toolkit (VLAT) <vlat.html>`_:
  The VERA suite includes a special set of tools known as the "VERA Locomotion Accessibility Toolkit" (VLAT), 
  which can help you provide locomotive and mobility accessibility options to participants with various physical disabilities. 
  This section provides a guide on setup and use of these VLAT tools.

Other important sections of the documentation include:

`Documentation Home <../index.html>`_:
  The home page of the documentation, which provides an overview of VERA and links to all major sections.
  It may be useful to return to this documentation home page to navigate to other sections.

`VERA Creed <creed.html>`_:
  The VERA Creed is a set of principles that guide the development and use of VERA, ensuring that it remains a tool for ethical and responsible research in XR.

`Quickstart <quickstart.html>`_:
  A quickstart guide that provides a brief overview of how to get started with VERA, including installation and setup instructions.
  Provides a much higher-level overview of the process of getting started with VERA, and is a good place to start if you are new to VERA or XR research in general.
